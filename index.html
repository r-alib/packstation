База — Google Таблица или файл .xlsx: по вкладке на платформу (заказы) + вкладка Inventory (остатки, расположение, фото). Каждый день — новая база.

Google Таблица тянуть по ссылке, без загрузки файла

В таблице: Поделиться → «Все, у кого есть ссылка» → Читатель. Колонку трекинга задайте формат «Простой текст».

или загрузите файл вручную

Колонки заказов: tracking id · item name · variation · quantity. Колонки Inventory: item name · variation · location · stock · barcode · image. Понимаются и русские названия колонок.
